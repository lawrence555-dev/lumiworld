# LumiWorld 先生と保護者のための指導ガイド

**LumiWorld** へようこそ！このガイドは、先生と保護者の皆様のために設計された包括的な教育ガイドです。LumiWorld は、**STEAM (科学、技術、工学、芸術、数学)** と **SEL (社会性と情動の学習)** を組み合わせた 8 週間のインタラクティブなカリキュラムです。3〜6 歳のお子様が世界を探索し、論理的思考を養い、環境を思いやる心を育むことを目的としています。

---

## 🌟 教育の核心目標

このシステムは単なる分類ゲームではありません。専門的な教育設計が組み込まれています。

1.  **適応型スキャフォールディング (足場かけ)**:
    - お子様の回答状況に応じてシステムが動的に調整します。困難に直面した場合、システムはアニメーションによるヒントを提供したり、誤った選択肢を無効にしたりして、達成感を維持しながら学習をサポートします。
2.  **多感覚学習**:
    - すべてのアイテムにはアイコン、テキスト、多言語音声が含まれており、視覚と聴覚の結びつきを強化します。
3.  **読書科学の基礎**:
    - 科学的探索の中に、基礎的な音素認識（解剖学ユニットでの文字と音の連結など）を統合しています。
4.  **データ駆動型フィードバック**:
    - 星による評価システムを通じて、先生や保護者はお子様の分類ロジックの習熟度を明確に把握できます。

## 🎮 ゲームの仕組みとデザインの詳細

### インタラクション
- **ドラッグ＆ドロップ**: 目と手の協調性と微細運動能力を鍛えます。
- **リアルタイムのポジティブ・フィードバック**: 正解すると紙吹雪と励ましの音が流れ、不正解の場合は優しく再試行を促します。

### インテリジェント・アシスト
- **2回のミス**: 正解のゾーンが点滅してガイドします。
- **3回のミス**: 不正解のゾーンが自動的にロックされ、お子様が正解に集中できるようにして、挫折感を防ぎます。

## 💡 先生と保護者へのアドバイス

### 学校で (先生)
1.  **グループ探索**: 2〜3 人のグループで、「なぜこれは生き物なの？」と話し合いながら進めることをお勧めします。
2.  **テーマの連動**: 週のテーマに合わせて、教室に関連するカードや標本を掲示してください（例：第6週に豆を育てる）。
3.  **振り返り**: 完了後、どのアイテムが一番意外だったか発表してもらうことで、表現力を養います。

### 家庭で (保護者)
1.  **生活との結びつき**: 「巨大 vs 微小」の週が終わったら、家の中で Huge なものと Tiny なものを一緒に探してみてください。
2.  **考えさせるサポート**: お子様が迷っているときは、すぐに答えを教えるのではなく、「猫ちゃんは大きくなるかな？」と問いかけてみてください。
3.  **多言語環境**: 言語設定を切り替えて、遊びの中で自然に外国語の音声に触れさせてみてください。
