## 🎯 專案概述

LumiWorld 是一款專為 4 歲兒童設計的互動式教育 PWA，透過拖放遊戲專注於 STEAM 教育。本專案完全對齊 **美國 Pre-K STEAM 與早期素養** 標準（CCSS 與 NGSS）。

**線上體驗:** [https://lumiworld.onrender.com](https://lumiworld.onrender.com)

---

## ✨ 核心功能

| 功能 | 說明 |
|------|------|
| **8 週完整課程** | 每個關卡 30 題，共 240 道專屬插畫練習題 |
| **AI 原創插圖** | 240+ 全新生成繪本風插圖 + 16 張類別引導場景圖 |
| **5 種語言支援** | 英文、繁體中文、日本語、한국어、ภาษาไทย |
| **頂級語音引擎** | 自然語音朗讀物品名稱 (含泰、日、韓語音專門修正) |
| **多語系教學指南** | 提供 5 種語言的詳細老師與家長教學指南 |
| **學習報告面板** | 一目瞭然的掌握度分析（時間統計、進度與能力狀態） |
| **iPad 極致優化** | 專為 iPad Air/Pro 設計的一頁式佈局，完美適配 PWA |
| **智慧拖放反饋** | 具備碰撞偵測與視覺引導的互動機制 |
| **進度持久化** | 星星評分系統 (1-3 星) + LocalStorage 無後端儲存 |
| **隱私與安全** | 無數據收集、無追蹤，符合 COPPA 兒童保護規範 |

## 📚 8 週課程總覽

每個關卡包含 **30 道練習題**，透過拖放分類學習：

| 關卡 | 主題 | 分類方式 | 核心標準 |
|:----:|------|----------|----------|
| **1** | 科學分類 | 有生命 vs 無生命 | NGSS K-LS1-1 |
| **2** | 解剖學入門 | 魚類構造 vs 其他部位 | 生物觀察 / 解剖學 |
| **3** | 數感發展 | 少量 vs 多量 | CCSS.MATH.CONTENT.K.CC.B.4 |
| **4** | 比較測量 | 巨大 vs 微小 | CCSS.MATH.CONTENT.K.MD.A.2 |
| **5** | 棲息地探索 | 天空 vs 地面 | NGSS 空間邏輯 |
| **6** | 植物學基礎 | 植物需要 vs 不需要 | NGSS 生命週期 |
| **7** | 環境科學 | 乾淨海洋 vs 垃圾 | NGSS K-ESS3-3 + CASEL SEL |
| **8** | 生態系統管理 | 森林 vs 海洋 | STEAM 整合 |

## 📁 專案結構

```
lumiworld-persistent-system/
├── app/                      # Next.js App Router (頁面與路由)
├── components/               # UI 與遊戲組件
├── systems/                 # 核心邏輯 (存檔、音訊、遊戲邏輯)
├── data/                    # GameContent.ts (240 道練習題)
├── hooks/                   # useProgress, useLanguage
├── locales/                 # 5 種語言翻譯檔 (i18n)
└── docs/                    # 多語系教師與家長指南
```
